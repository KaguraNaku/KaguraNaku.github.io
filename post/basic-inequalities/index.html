【初等代数】基本不等式

导语

基本不等式是利用比较方法快速推断出不同代数式的大小关系的重要工具。横跨初等代数，高等代数。虽然基础，但并不是所有的基本不等式都易于记忆。本文旨在从原理及应用角度深入分析这些较难理解的基本不等式，帮助读者真正理解，跳出仅会做题却不知其所以然的误区。

正篇

均值不等式

称不等式 $\dfrac{2}{\dfrac{1}{a} + \dfrac{1}{b}} \leqslant \sqrt{ab} \leqslant \dfrac{a + b}{2} \leqslant \sqrt{\dfrac{a^2 + b^2}{2}} \quad (a, b > 0)$ 为均值不等式。其中：

  • $\dfrac{2}{\dfrac{1}{a} + \dfrac{1}{b}}$ 为调和平均数
  • $\sqrt{ab}$ 为几何平均数
  • $\dfrac{a + b}{2}$ 为算数平均数
  • $\sqrt{\dfrac{a^2 + b^2}{2}} \ (a, b > 0)$ 为平方平均数

记忆

Conclusion

（注：以下称呼均为省略“平均数”的简称。）均值不等式满足这样的关系：调和 $\le$ 几何 $\le$ 算术 $\le$ 平方。在吃透原理后，这样的记忆比记抽象的代数组合要方便的多。当然，前提是——“吃透”。

原理

Important

我们需要首先明白推导的顺序。先有 几何平均数 $\to$ 算术平均数，即$\sqrt{ab} \leqslant \dfrac{a+b}{2}$。再在这个的基础上得出均值不等式的其他部分。正确的推导顺序是顺利得到均值不等式的大前提。

  1. 对于几何平均数 $\to$ 算术平均数， 即$\sqrt{ab} \leqslant \dfrac{a+b}{2}$，证明流程如下。
  • 构建完全平方差公式，$(\sqrt{a} - \sqrt{b})^{2}=a - 2\sqrt{ab} + b$。
  • 完全平方式的结果非负，所以有$a - 2\sqrt{ab} + b \geqslant 0$。
  • 移项得到$\dfrac{a+b}{2} \geqslant \sqrt{ab}$。$(1)$
  • 得证。
  1. 对于调和平均数 $\to$ 几何平均数，即$\dfrac{2}{\dfrac{1}{a} + \dfrac{1}{b}} \leqslant \sqrt{ab}$，证明流程如下。
  • 将 $(1)$ 式中的a,b取倒数后复代入 $(1)$ 式，得：$\sqrt{\dfrac{1}{ab}} \leqslant \dfrac{\dfrac{1}{a} + \dfrac{1}{b}}{2}$。
  • 两边同取倒数，得$\sqrt{ab} \geqslant \dfrac{2}{\dfrac{1}{a} + \dfrac{1}{b}}$。
  • 得证。
  1. 对于算术平均数 $\to$ 平方平均数，即$\dfrac{a+b}{2} \leqslant \sqrt{\dfrac{a^2+b^2}{2}}$，证明流程如下。
  • 两边平方有理化，得到$(\dfrac{a+b}{2})^2 \leqslant \dfrac{a^2+b^2}{2}$。对右式通分得到$(\dfrac{a+b}{2})^2 \leqslant \dfrac{2a^2 + 2b^2}{4}$。
  • 移项作差，得到$\dfrac{-a^2+2ab-b^2}{2^2} \leqslant 0$，变号作完全平方差得$\dfrac{a^2-2ab+b^2}{2^2}=(\dfrac{a-b}{2})^2 \gt 0$恒成立。
  • 得证。

AM-GM不等式拓展

在均值不等式中，称 $\sqrt{ab} \leqslant \dfrac{a + b}{2} $ 为AM-GM不等式。介于AM-GM不等式的二元形式已在前给出证明，这里仅提供其拓展到三元的证明。

  1. 三元AM-GM不等式定义为：$\sqrt[3]{abc} \le \dfrac{a+b+c}{3}$， 其中$a,b,c \ge 0$。 $(1)$ （若 $a,b,c$ 其中任一项或多项为负，其在右式中的情况将难以预测）
  2. 在平凡情况中，$a,b,c$ 中任一项为 $0$ ，易证成立。
  3. 若 $a,b,c \gt 0$ ，不妨设 $g = \sqrt[3]{abc} ; (g \gt 0)$ 。则AM-GM不等式可以记作：$\dfrac{a+b+c}{3} \ge g \Rightarrow \dfrac{a+b+c}{3g} \ge 1 \Rightarrow \dfrac{a+b+c}{g} \ge 3$ 。 $(2)$
  4. 设 $x=\dfrac{a}{g}, y=\dfrac{b}{g}, z=\dfrac{c}{g} ; (x,y,z \gt 0) \ $。则 $(2)$ 式可以写作 $x+y+z \ge 3$。即需证: $x+y+z \ge 3$。 $(3)$
  5. 注意到 $xyz = \dfrac{abc}{g^3} = \dfrac{abc}{(\sqrt[3]{abc})^3} = \dfrac{abc}{abc} = 1$ 。 $(4)$
  6. 由 $(4)$ 式得： $z=\dfrac{1}{xy}$。由 $(3)$ 式得：$x + y + \dfrac{1}{xy} \ge 3$。 $(5)$
  7. 对 $(5)$ 式中的 $x+y$ 应用二元AM-GM不等式有 $x+y \ge 2\sqrt{xy}$ 。不妨记 $t=\sqrt{xy} ; (t \gt 0)$ ，则问题转化为证：$2t+\dfrac{1}{t^2} \ge 3$。 $(6)$
  8. 对 $(6)$ 式有：$\dfrac{1}{t^2} + 2t - 3 \ge 0 \Rightarrow \dfrac{2t^3-3t^2+1}{t^2}=\dfrac{(t-1)^2(2t+1)}{t^2} \ge 0$。得证。

三角不等式

称不等式 $||a| - |b|| \leqslant |a + b| \leqslant |a| + |b|$ 为三角不等式。其中：

  • $||a| - |b|| = |a + b|$ 在 $a \cdot{b} \le 0$ 的时候成立。
  • $|a + b| = |a| + |b|$ 在 $a \cdot{b} \ge 0$ 的时候成立。

原理

Important

类似均值不等式，同样需要着重推导的顺序。“先后再前”。

  1. 对于 $|a + b| \le |a| + |b|$，证明流程如下。
  • 对两边平方，左式得到$|a + b|^2 = (a + b)^2 = a^2 + 2ab + b^2$。右式得到$(|a| + |b|)^2 = |a|^2 + 2|a||b| + |b|^2 = a^2 + 2|ab| + b^2$。
  • 又 $ab \le |ab|$（当且仅当 $ab \ge 0$ 的时候等号成立），得：$(a + b)^2 \le (|a| + |b|)^2$。对两边开方，得$|a + b| \le |a| + |b|$。$(1)$（当且仅当 $ab \ge 0$ 的时候等号成立）
  • 得证。
  1. 对于 $\big||a| - |b|\big| \le |a + b|$，证明流程如下。
  • 设 $|a| \ge |b|$。反之，若设 $|b| \ge |a|$，则 $\big||a|-|b|\big| \Rightarrow \big||b|-|a|\big|$。
  • 对于$|a|$，构造与右边等式相同形式，有$|a| = |(a+b)-b|$。
  • 利用 $(1)$ 式结论，即 $|a+b| \le |a|+|b|$，令 $b=-b$，得 $|a-b| \le |a|+|b|$。
  • 则对 $|a| = |(a+b)-b|$，令 $b=-b$，得$|a| = |(a-b)+b| \le |a-b|+|b|$。（利用原 $(1)$ 式结论）
  • 复令 $b = -b$，得$|a| = |(a+b)-b| \le |a+b|+|b|$。则 $\big||a|-|b|\big| \le |a+b|$。
  • 得证。

柯西不等式

称 $(a_1^2 + a_2^2 + \cdots + a_n^2)(b_1^2 + b_2^2 + \cdots + b_n^2) \geqslant (a_1b_1 + a_2b_2 + \cdots + a_nb_n)^2$ 为柯西不等式。

原理

此处我们提供三种方法。从不同位面深入分析柯西不等式。

数学归纳法
  1. 当 $n=2$ 时，柯西不等式可写为：$(a_{1}^2 + a_{2}^2)(b_{1}^2+b_{2}^2) \ge (a_{1}b_{1} + a_{2}b_{2})^2$。
  • 对左式，$(a_{1}^2 + a_{2}^2)(b_{1}^2+b_{2}^2) = a_{1}^2b_{1}^2+a_{1}^2b_{2}^2 + a_{2}^2b_{1}^2 + a_{2}^2b_{2}^2$。
  • 对右式，$(a_{1}b_{1} + a_{2}b_{2})^2 = a_{1}^2b_{1}^2 + 2a_{1}a_{2}b_{1}b_{2} + a_{2}^2b_{2}^2$。
  • 作差，得： $$ \begin{align*} &\quad (a_1^2 + a_2^2)(b_1^2 + b_2^2) - (a_1b_1 + a_2b_2)^2 = a_1^2b_1^2 + a_1^2b_2^2 + a_2^2b_1^2 + a_2^2b_2^2 - (a_1^2b_1^2 + 2a_1a_2b_1b_2 + a_2^2b_2^2) \\ &= a_1^2b_2^2 + a_2^2b_1^2 - 2a_1a_2b_1b_2 \\ &= (a_1b_2 - a_2b_1)^2 \ge 0 \end{align*} $$
  • 得证：当 $n=2$ 时， $(a_{1}^2 + a_{2}^2)(b_{1}^2+b_{2}^2) \ge (a_{1}b_{1} + a_{2}b_{2})^2$ 成立。

  1. 设当 $n=k(k\ge2)$ 时，命题 $\sum_{i=1}^k{a_i}^2\sum{b_{i}^2} \ge (\sum_{i=1}^k{a_{i}b_{i})}^2$成立。 即 $(a_{1}^2+a_{2}^2+\cdots+a_{k}^2)(b_{1}^2+b_{2}^2+\cdots+b_{k}^2) \ge (a_{1}b_{1}+a_{2}b_{2}+\cdots+a_{k}b_{k})^2$成立。

  2. 当 $n=k+1$ 时，需证式 $\sum_{i=1}^{k+1}{a_i}^2\sum{b_{i}^2} \ge (\sum_{i=1}^{k+1}{a_{i}b_{i})}^2$ 成立。

    • 先对左式做计算。
    • $\sum_{i=1}^{k+1}{a_i}^2\sum{b_{i}^2} = (\sum_{i=1}^k{a_i}^2+a_{k+1}^2)(\sum{b_{i}^2}+b_{k+1}^2)= \sum_{i=1}^ka_{i}^2\sum{b_{i}^2} + b_{k+1}^2\sum_{i=1}^ka_{i}^2+a_{k+1}^2\sum_{i=1}^{k}{b_{i}^2}+a_{k+1}^2b_{k+1}^2$ 。$(1)$
    • 由归纳假设，对$(1)$式有 $\sum_{i=1}^{k+1}{a_i}^2\sum{b_{i}^2} \ge (\sum_{i=1}^k{a_{i}b_{i})}^2 + b_{k+1}^2\sum_{i=1}^ka_{i}^2+a_{k+1}^2\sum_{i=1}^{k}{b_{i}^2}+a_{k+1}^2b_{k+1}^2$。$(2)$
    • 再计算右边式子。
    • $(\sum_{i=1}^{k+1}{a_{i}b_{i})}^2 = (\sum_{i=1}^{k}{a_{i}b_{i}+a_{k+1}b_{k+1}})^{2}=(\sum_{i=1}^{k}{a_{i}b_{i}})^2 + 2a_{k+1}b_{k+1}\sum_{i=1}^{k}{a_{i}b_{i}}+(a_{k+1}b_{k+1})^2$。$(3)$
    • 左右两式作差，$(2)-(3)$ 得：
    • $b_{k+1}^2\sum_{i=1}^ka_{i}^2 - 2a_{k+1}b_{k+1}\sum_{i=1}^{k}{a_{i}b_{i}} + a_{k+1}^2\sum_{i=1}^{k}{b_{i}^2}=\sum_{i=1}^k(b_{k+1}^2a_{i}^2 - 2a_{k+1}b_{k+1}{a_{i}b_{i}} + a_{k+1}^2{b_{i}^2})$。$(4)$
    • 需证 $(4)$ 式非负。作完全平方，得：
    • $\sum_{i=1}^k(b_{k+1}^2a_{i}^2 - 2a_{k+1}b_{k+1}{a_{i}b_{i}} + a_{k+1}^2{b_{i}^2})=\sum_{i=1}^k(a_{i}b_{k+1}-b_{i}a_{k+1})^2 \ge 0$。$(5)$
    • 递推得证：式 $\sum_{i=1}^{k+1}{a_i}^2\sum{b_{i}^2} \ge (\sum_{i=1}^{k+1}{a_{i}b_{i})}^2$ 成立。

  3. 得证：$\sum_{i=1}^{k+1}{a_i}^2\sum{b_{i}^2} \ge (\sum_{i=1}^{k+1}{a_{i}b_{i})}^2$ ，即$(a_1^2 + a_2^2 + \cdots + a_n^2)(b_1^2 + b_2^2 + \cdots + b_n^2) \geqslant (a_1b_1 + a_2b_2 + \cdots + a_nb_n)^2$ 成立。

构造函数法

注意到$(a_1^2 + a_2^2 + \cdots + a_n^2)(b_1^2 + b_2^2 + \cdots + b_n^2) \geqslant (a_1b_1 + a_2b_2 + \cdots + a_nb_n)^2$ 描述的是二次多项式间的一种关系，不妨构造二次函数，并令其恒非负。此处的非负性并不是为了类似作差那样的比较，而是利用判别式 $\Delta$ 的性质。

  1. 构造二次函数$f(t) = { (\sum_{i=1}^n{a_{i}}^2){t^{2}} } + { (2\sum_{i=1}^{n}{a_{i}b_{i}})t } + { \sum_{i=1}^{n}b_{i}^2 } = { (\sum_{i=1}^{n}{a_i} + \sum_{i=1}^{n}{b_i})^2 } \ge {0}$，则在 $t$ 的定义域上，$\Delta \le 0$。
  2. $\Delta = { {(2\sum_{i=1}^{n}{a_{i}b_{i}})}^2 } - { { 4(\sum_{i=1}^n{a_{i}}^2) }({ \sum_{i=1}^{n}b_{i}^2 }) } \le 0$。移项并约去公因式，得${ ({ \sum_{i=1}^{n}a_{i}b_{i})^2 } \le { (\sum_{i=1}^n{a_{i}}^2) }{ (\sum_{i=1}^{n}b_{i}^2) } }$ 成立。
  3. 得证。
向量内积法

不妨将 ${{\boldsymbol{a_n}}}, {\boldsymbol{b_n}}$ 视为向量。则对于原式 $\sum_{i=1}^{n}{a_i}^2\sum{b_{i}^2} \ge (\sum_{i=1}^{n}{a_{i}b_{i})}^2$ 有：

  1. 模长 $|\boldsymbol{a}| = \sqrt{\sum_{i=1}^{n}a_{i}^{2}},; |\boldsymbol{b}| = \sqrt{\sum_{i=1}^{n}b_{i}^{2}}$。向量内积 $\boldsymbol{a} \cdot \boldsymbol{b} = |a||b|\cdot cos\theta$ ，且$\boldsymbol{a} \cdot \boldsymbol{b} = \sum_{i=1}^{n}a_{i}b_{i}$。
  2. 平方后的不等式等价于： $\left(\big||\boldsymbol{a}| \cdot |\boldsymbol{b}|\big|\right)^2 \ge |\boldsymbol{a} \cdot \boldsymbol{b}|^2 = \left(|\boldsymbol{a}| \cdot |\boldsymbol{b}| \cdot |\cos\theta|\right)^2$。又 $|cos\theta| \le 1$，得证不等式成立。
Warning

常见错误：下意识将模长计算带入绝对值计算法则。如此，会得到: $|a||b| = |ab|$。这是严重谬误！！！

Conclusion

柯西不等式是连通代数和几何的重要工具，从数学归纳法 $\to$ 构造函数法 $\to$ 向量内积法 体现了从代数到几何的三段式跨越。介于本文范畴限定在初等代数，有关于其正定性质等在后续文章中再详细叙述。

总结

对这三个较难直接记忆的不等式，采用了诸如直接作差比较或构造恒正、恒负函数等方式进行证明。作差是最直接了当的比较型，而构造函数则更为精巧，利用构造函数的性质以求解不等式问题。如利用二次函数处理柯西不等式中的二次型关系，利用向量内积处理向量关系等。
